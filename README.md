# Darkroom Tools

A progressive web app with five calculators for analog photography darkroom work. Works offline once installed.

## Tools

1. **Exposure Time Calculator** — Adjust exposure when changing contrast filter grades. Supports Foma, Ilford, and Meopta Meochrom color head filter settings.
2. **F-Stop Timer** — Generate an f-stop timing table with configurable step sizes (1, 1/2, 1/3, 1/4, 1/6 stop) and dry-down compensation.
3. **Enlarger Exposure** — Magnification-based exposure correction when changing print size or enlarger height.
4. **Optical Print Placement** — Visual tool for centering and positioning a print within an easel/mount, with image overlay support.
5. **Opemus 5 Nomogram** — Exposure correction using the Meopta Opemus 5 manual nomogram for f=50-80 mm lenses, with interactive calibration chart.

## Usage

Open `index.html` in any modern browser, or visit: https://gradusnikov.github.io/darkroom-tools/

### Install as an app

On mobile or desktop, use your browser's "Add to Home Screen" or "Install App" option. The app works fully offline after the first load.
